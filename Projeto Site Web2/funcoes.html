Funções Anonimas

Não apresenta nome.
Devem ser atribuidas a uma variável para serem invocadas;

No exemplo a baixo o que você digitar no 'input' será exibido como mensagem para o alert.

 var anonima =  function() {
	'use strict';
	var texto = document.getElementById("input-text").value;
	return window.alert(texto);
 };

 document.getElementById("testeAnonima").onclick = anonima;

Função Aninhada

Funções aninhadas são aquelas que possuem um outra função declarada internamente.
Função interna apenas é acessivel dentro da função externa.

No exemplo abaixo a função 'barulho' está interna à função 'funcCachorro'.

 function funcCachorro(nome) {
	 'use strict';
	 nome = document.getElementById("texto_aninhada").value;

	 function barulho(nome) {
	 	return nome + " late!";
	 }
	 document.getElementById("resultado_aninhada").innerHTML = (barulho(nome));
 }

 document.getElementById("bt_aninhada").onclick = funcCachorro;
