JQuery

JQuery é uma biblioteca de funções JavaScript desenvolvida para simplificar a manipulção de documentos HTML, eventos, animações e interações AJAX.

A sintaxe para executar com jQuery é: $(selector).ação() ou jQuery(selector).ação()
O arquivo jquery-3.2.1.min.js precisa estar referenciado no projeto ou o cdn “http://ajax.googleapis.com/ajax/libs/ jquery/2.1.4/jquery.min.js".

Exemplos

Clique nos botões ao lado e veja o que acontece com os personagens.

window.onload = function(){
    $('#div-homer-bart').hide();
}
                                
$('#btn-esconda-pai').click(function(){
    $('#div-pai').fadeOut(2000);
});
                                
$('#btn-apresente-pai').click(function(){
    $('#div-pai').fadeIn(2000);
    $('#div-homer-bart').hide();
});
                                
$('#btn-esconda-filho').click(function(){
    $('#div-filho').fadeOut(2000);
});
                                
$('#btn-apresente-filho').click(function(){
    $('#div-filho').fadeIn(2000);
    $('#div-homer-bart').hide();
});
                                
$('#btn-pai-filho').click(function(){
    $('#div-filho').hide();
    $('#div-pai').hide();
    $('#div-homer-bart').fadeIn(4000);
});

Perceba que o código para execução das funções ficou muito menor e simplificado.
